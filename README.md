# Alsouq Poultry Project

Application d'administration pour Alsouq Poultry avec architecture monorepo.

## Structure du projet

```
alsouq-poultry-project/
├── frontend/          Application React (Vite)
├── backend/           API Backend
├── README.md
└── .gitignore
```

## Installation

### Frontend

```bash
cd frontend
npm install
npm start
```

Le frontend sera accessible sur `http://localhost:3000`

### Backend

```bash
cd backend
# Suivez les instructions d'installation de votre backend
```

## Technologies

- **Frontend**: React 19, Vite, CoreUI, React Router, Redux
- **Backend**: (à compléter selon votre stack)

## Développement

### Frontend
- Port: 3000
- Proxy API: `/api/*` → `https://localhost:7054`

### Backend
- Port: 7054 (HTTPS)
